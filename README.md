아무 메모나 자유롭게 적을 수 있는 싱글 페이지입니다.

일을 하다가 머릿 속에 떠오르는 생각들을 정리할 수 있습니다.

회의를 하면서 회의 내용을 받아 적는데도 사용할 수도 있습니다.

**LocalStorage**를 사용했기 때문에, 창을 닫아도 메모 내용이 사라지지 않습니다. 다만, 다른 기기에서 내용을 불러올 수는 없습니다.
